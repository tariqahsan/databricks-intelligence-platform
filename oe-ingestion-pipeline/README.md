# OE Data Intelligence — Mock Log Ingestion Pipeline

## What This Does

```
generate_mock_logs.py
        │
        ▼  Creates realistic raw sys logs
┌───────────────────────────────────────────────────────┐
│  mock-logs/                                           │
│    aternity/   → JSONL  App performance per session   │
│    netscout/   → CSV    Network flows per segment     │
│    intune/     → JSONL  Device snapshots              │
│    infoblox/   → Syslog DNS query logs (RFC 5424)     │
│    salesforce/ → CSV    Incident tickets              │
│    sciencelogic/→ JSONL Infrastructure alerts         │
└───────────────────────────────────────────────────────┘
        │
        ▼  Spark reads raw files
    BRONZE (Delta tables in MinIO S3)
        │
        ▼  Cleanse + validate
    SILVER (Delta tables in MinIO S3)
        │
        ▼  Aggregate into product KPIs
    GOLD   (Delta tables — queried by Spring Boot API)
```

## Quick Start

### Step 1 — Start Docker stack
```bash
cd oe-ingestion
docker compose up -d

# Wait for Jupyter to start
docker compose logs -f jupyter
# → "Jupyter Server is running at: http://localhost:8888"
```

### Step 2 — Generate mock logs
```bash
# Option A: inside Jupyter terminal
python /home/jovyan/scripts/generate_mock_logs.py --days 3

# Option B: on your host machine
python scripts/generate_mock_logs.py --days 3
```

**Output:**
```
✅ Aternity:     7,500 records  (JSONL)
✅ NetScout:     9,000 records  (CSV)
✅ Intune:       1,200 records  (JSONL)
✅ Infoblox:    24,000 records  (Syslog)
✅ Salesforce:      20 records  (CSV)
✅ ScienceLogic: 4,500 records  (JSONL)
```

### Step 3 — Open Jupyter and run pipeline
```
http://localhost:8888  (password: databricks)
→ Open: notebooks/oe_pipeline_walkthrough.ipynb
→ Run All Cells
```

Or run from command line inside Jupyter terminal:
```bash
cd /home/jovyan
python pipelines/run_pipeline.py
```

### Step 4 — Start Spring Boot in mock-free mode
With real Gold tables now populated, start without the mock profile:
```bash
cd ../backend
mvn spring-boot:run -Dspring-boot.run.profiles=local
```

The API at `http://localhost:8091` now reads real Delta data.

## Issues Simulated Per Source

### Aternity (app_performance)
- Microsoft Teams: experience score ~38, 1800ms response, crashes
- Outlook Web App: score ~45, sync failures, OWA_TIMEOUT errors
- ServiceNow: score ~60, DB connection pool exhaustion
- All others: healthy baseline with realistic variance

### NetScout (network_metrics)
- DISA-WAN-DC-01: 3.8% packet loss (CONGESTION root cause)
- DISA-WAN-PENTAGON: 2.9% loss (HARDWARE_DEGRADATION)
- DISA-WAN-MEADE-01: 2.1% loss (LINK_SATURATION)
- Peak hour traffic spikes (8am-6pm EST)
- 6 root causes with AI confidence scores

### Intune (device_inventory)
- Windows 7 SP1 / Windows 8.1: always NON_COMPLIANT
- iOS 16.7 / Android 13: NON_COMPLIANT
- 12% random non-compliance on modern devices
- Devices not checked in > 7 days flagged

### Infoblox (dns_metrics)
- Primary server: 380ms avg response (normal: 2ms)
- NXDOMAIN for misspelled domains
- REFUSED for suspicious C2/malware domains
- SERVFAIL during simulated primary failure

### Salesforce (incidents)
- 2 P1 open: Teams failure + WAN packet loss
- 7 P2: Pentagon hardware, Intune MDM, DNS, Win7 STIG
- Full MTTR tracking, SLA breach flags
- Recurring issue patterns

### ScienceLogic (infrastructure_alerts)
- Router CPU/interface alerts tied to NetScout issues
- DB connection pool, SMTP queue, VPN SA failures
- Security alerts: IDS signature match, firewall blocks
- Storage capacity warnings

## File Structure
```
oe-ingestion/
  docker-compose.yml
  scripts/
    generate_mock_logs.py     ← Run this first
  pipelines/
    run_pipeline.py           ← Master orchestrator
    bronze/
      ingest_all_sources.py   ← Reads raw files → Bronze Delta
    silver/
      transform_all.py        ← Cleanses → Silver Delta
    gold/
      aggregate_all.py        ← Aggregates → Gold Delta (API-ready)
  notebooks/
    oe_pipeline_walkthrough.ipynb ← Interactive walkthrough
  mock-logs/                  ← Generated by generate_mock_logs.py
    aternity/   netscout/   intune/
    infoblox/   salesforce/ sciencelogic/
```
